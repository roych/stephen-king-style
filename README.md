# Stephen King Style — 斯蒂芬·金写作风格引擎

通过 `/stephen-king` 显式调用，让 AI 以斯蒂芬·金的文字风格完成以下任务。

## 三种模式

| 模式 | 说明 | 用法示例 |
|------|------|----------|
| **Writing** | 按金式风格写新内容 | `/stephen-king 写一段地下室里有怪声音的场景` |
| **Rewriting** | 把已有文本改写成金式风格 | `/stephen-king 把这段话改成金式风格：[贴文本]` |
| **Analyzing** | 诊断文本的金式特征覆盖率 | `/stephen-king 分析这段话有多像斯蒂芬·金：[贴文本]` |

## 风格基础

基于 150+ 段热门划线、14 部作品分析，提取 10 维特征：
5 条核心铁律（必须遵守）+ 5 条进阶技法（按场景选用）。
